# GRIHT

**Put In Work** — Workout tracker built for the gym.

## Stack
- React 18 + Vite
- Outfit + JetBrains Mono (Google Fonts)
- localStorage persistence
- PWA-ready (add to homescreen)

## Dev
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → Connect to Vercel → Auto-deploys on push.

## Domain
Configure `griht.com` in Vercel dashboard → Settings → Domains.
